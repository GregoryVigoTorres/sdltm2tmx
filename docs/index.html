sdltm2tmx

Get the code on github

Sdltm2tmx is a cli app and library written in Python to convert .sdltm files into tmx 1.4 translation memories that can be used with OmegaT or other applications.

Most translation meta data (creation date, creation id etc.) is preserved in the converted tmx file. However, sdltm2tmx is still in beta and I think some sdltm files might not work yet. Luckily, it's free software so contributions and bug reports are welcome.
